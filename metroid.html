Metroid 1986 - Unity Remaster

About the Project:

This is a Unity remaster of the first realm of the classic 1986 game Metroid, with a customized extra level that features a new mechanic.

My Contributions:

Remade the first realm of the NES video game Metroid and added a custom level to show off a custom mechanic. Created with Tianze You.

My contributions: Implemented the entire player (Samus), implemented every enemy (Zoomers, Reos, Skrees, Rippers, and Zebs), implemented the UI, and created the custom level layout

The custom mechanic is the ability to walk on the celling with Samus's new Anti-Gravity Suit. Race through a custom made level to show off your physics-defining abilities.

Technology Used:

The remaster was developed using the Unity Game Engine, powered by the programming language C#. The WebGL build of the game is hosted on itch.io.
