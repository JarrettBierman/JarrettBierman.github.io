Golfpocalypse

About the Project:

This is a rapid prototype of a golf simulation / survival adventure video game. It was made as a project for EECS 494 in the Winter of 2023.

My Contributions:

With few exceptions, everything was made by me. A few notable exceptions include the Zombie prefab and animations, two sound effects, and some grass textures.

Everything else, from the visuals to the mechanics to the music, was created by me in a two week time period.

Technology Used:

This game was developed using the Unity Game Engine, powered by the programming language C#. You can download the game for Mac or Windows below...
